# Verma Labs

Experimental software lab exploring:

- computer graphics
- AR / VR
- generative systems
- human-computer interaction
- real-time simulations

Projects are built as part of **Project52** — a challenge to build one technical project every week.

## Current stack

Python • OpenGL • Computer Vision • AR • Procedural Generation

## Projects

- Galaxy Explorer — procedural galaxy simulation
- ASCII Camera — real-time ASCII video renderer
- AR Model Viewer — markerless 3D object placement
- Hand-Gesture Interface — camera-based input system
